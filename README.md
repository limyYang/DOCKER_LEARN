# my_docker
+++ centos7（或者更高版本下）
## **docker安装**

`yum -y update` ：更新一波yum  <br/>
`yum install docker -y` ：yum下载安装docker <br/>

## **docker安装完毕后**
`doker pull centos:7.6.1810`:下载centos镜像版本伟7.6.7.1810（默认伟最新的） <br/>
`doker images`: 查看docker中的所有镜像 <br/>
`doker rmi`: 删除镜像 <br/>
`doker search -s 10`: 查询star数目超过10的镜像 <br/>
